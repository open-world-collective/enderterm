# License Choice: GPL vs AGPL

## Recommendation

Use **AGPL-3.0-only** as the default for this tooling stack.

## Rationale

- GPL-3.0 covers redistribution of binaries/source.
- AGPL-3.0 additionally requires source availability for network-served modifications.
- This supports openness as tooling evolves into service-like infrastructure.

## Caveat

This document is an engineering recommendation, not legal advice.
